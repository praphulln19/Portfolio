## 🛠️ Built With
- **React** (v19) - Core framework
- **Vite** - Lightning-fast development server & bundler
- **Tailwind CSS v4** - Modern utility-first CSS framework configured with custom variants
- **Framer Motion** - Library for rich viewport and hover animations
- **Lucide React** - Clean SVG icon set

## ✨ Key Features
- **Centralized Data Structure:** All content (skills, projects, education) is managed through a single `portfolioData.js` object, simulating a robust backend structure and making future updates effortless.
- **Premium Aesthetics:** Features a modern glassmorphism design approach with customized CSS variables and layered blurs.
- **Custom Theming:** Native Dark/Light mode integration hooked seamlessly into Tailwind v4.
- **Typography:** Configured explicitly with **Space Grotesk** for a striking, technical visual identity.

## 🚀 Getting Started

Follow these steps to run the portfolio locally on your machine.

### Prerequisites
You will need Node.js installed. 

### Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/praphulln19/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   *Your portfolio should now be running at `http://localhost:5173`.*
